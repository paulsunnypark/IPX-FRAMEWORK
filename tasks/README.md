# 작업 관리 디렉토리

이 디렉토리는 Poly Vibe Coding Method에 따라 프로젝트의 작업과제를 관리합니다.

## 파일 구조

### 핵심 문서
- **`context-contraction.md`**: 프로젝트 상태 추적 로그 (SSOT)
  - 모든 변경사항과 결정사항을 기록
  - 다음 작업 시 반드시 참조

### PRD (Product Requirements Document)
- **`prd-demo-data-integration.md`**: 데모 데이터 통합 요구사항
- **`prd-template.md`**: PRD 작성 템플릿

### 작업 계획
- **`tasks-prd-demo-data-integration.md`**: 데모 데이터 통합 작업 계획
- **`tasks-template.md`**: 작업 계획 작성 템플릿

## 현재 진행 중인 작업

### 데모 데이터 통합
- **PRD**: `prd-demo-data-integration.md`
- **작업 계획**: `tasks-prd-demo-data-integration.md`
- **상태**: 계획 수립 완료, Phase 1 시작 대기

## 다음 작업 시작하기

1. **최신 상태 확인**: `context-contraction.md` 읽기
2. **작업 계획 확인**: `tasks-prd-*.md` 읽기
3. **실행**: `실행!` 또는 `실행! N.M` 명령 사용
